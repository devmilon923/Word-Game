How to Play

  1. Two players must join using the same Room Name.
  2. Each player takes turns entering an English word.
  3. The next word must start with the last letter of the previous word.
  4. Each valid English word gives you +5 points.
  5. Each invalid or misspelled word deducts 5 points.
